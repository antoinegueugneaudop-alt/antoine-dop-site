## Local preview

Open `index.html` directly in Safari/Chrome, or run:

```bash
python3 -m http.server 5173
```

Then open:

```text
http://localhost:5173
```

## Deployment

Fastest route: connect the GitHub repository to Vercel.

Vercel settings:

- Framework Preset: Other
- Build Command: leave empty
- Output Directory: leave empty / root
- Production Branch: `main`

## Important visual rule

Use only real stills / photograms supplied in this repository. Do not invent, relight, smooth, reconstruct, or AI-enhance work images.
